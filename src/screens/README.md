# Screens

앱의 각 화면 컴포넌트들을 모아놓는 폴더입니다.

## 구조

```
screens/
├── auth/           # 로그인/회원가입 화면
├── home/           # 홈 화면
├── patterns/       # 도안 관련 화면
├── profile/        # 프로필 화면
├── yarns/          # 실 정보 화면
└── needles/        # 바늘 정보 화면
```

## 화면 컴포넌트 작성 가이드

- 하나의 화면 = 하나의 컴포넌트
- 네비게이션 props 활용
- 상태 관리는 Redux 사용
- 로딩 및 에러 상태 처리
