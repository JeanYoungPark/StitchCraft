# Components

재사용 가능한 UI 컴포넌트들을 모아놓는 폴더입니다.

## 구조

```
components/
├── common/         # 공통 컴포넌트
├── forms/          # 폼 관련 컴포넌트
├── buttons/        # 버튼 컴포넌트
├── cards/          # 카드 컴포넌트
└── modals/         # 모달 컴포넌트
```

## 컴포넌트 작성 가이드

- 재사용 가능하도록 설계
- PropTypes 또는 TypeScript 사용
- 기본 props 제공
- 스타일은 StyleSheet 사용
